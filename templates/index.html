Description

This is the dummy web application for the MTS SHAD 23/24 ML-OPS homework. The aim of this homework is to understand how to implement ML models into production applications and how to manipulate them using Docker. In order to make the prediction you have to pass the csv file with user's data (example is here). After pushing predict button you will be able to download:
  • input data you passed in the csv format
  • submission csv file
  • top-5 feature importance JSON file
  • distribution of the predicted labels in the png format
